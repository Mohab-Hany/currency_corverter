# Currency Converter 💱

تطبيق سطح مكتب بيحول العملات بشكل لحظي باستخدام API. المستخدم يختار العملة المصدر والعملة الهدف والبرنامج بيجيب السعر الحالي ويحسب الناتج.

## 📌 المميزات
- دعم اكتر من 160 عملة حول العالم
- جلب اسعار الصرف لحظياً من API
- واجهة رسومية بسيطة باستخدام Tkinter
- عرض سعر الصرف وتاريخ اخر تحديث

## 🛠️ الأدوات المستخدمة
- `Python 3`
- `Tkinter` لبناء الواجهة
- `Requests` للتعامل مع API
- `ExchangeRate-API` للاسعار
