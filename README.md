# Currency Converter 💱

تطبيق بايثون بيحول بين العملات باستخدام اسعار الصرف اللحظية من API.

## 📌 المميزات
- تحويل لحظي للعملات باستخدام `apilayer Fixer API`
- دعم العملات: USD, EUR, EGP, SAR, AED, GBP, JPY, CAD
- التحقق من صحة العملة والمبلغ المدخل
- واجهة بسيطة عن طريق التيرمنال

## 🛠️ المتطلبات
- Python 3
- مكتبة requests
- API KEY من apilayer.com
