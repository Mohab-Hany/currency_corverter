# Website Scraper 🕷️

برنامج بيجيب البيانات من المواقع تلقائياً. بتديله رابط الصفحة وهو يستخرج العناوين والاسعار او اي داتا انت عايزها ويحفظها.

## 📌 المميزات
- استخراج البيانات من اي موقع
- يستخدم BeautifulSoup لتحليل HTML
- كود منظم وسهل التعديل
- يقدر يحفظ النتائج في ملف

## 🛠️ الأدوات المستخدمة
- `Python 3`
- `Requests` لجلب صفحات الويب
- `BeautifulSoup4` لتحليل HTML
- `lxml` parser
